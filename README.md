# 🍃 Your Serenity Companion
> *A gentle AI companion for your peace and knowledge.*
 
---

## 📌 About
 
Serenity Companion is a locally hosted AI chatbot web application built with JavaScript, Python, and Flask. It runs AI inference entirely on-device using Ollama. Also which can answer you depends on your mood and help you calm with a builtin breathing tracker.

---

## ⚙️ Technologies Used
 
| Technology | Purpose |
|---|---|
| JavaScript | Frontend interactivity and chat logic |
| Bootstrap 5 | UI styling via CDN |
| Python 3 | Backend runtime |
| Flask | Lightweight web server framework |
| Ollama | Local AI model runner |
| DeepSeek-R1:8b | AI language model |
